Password Manager 2FA — Full-stack

Gestionnaire de mots de passe avec authentification double facteur. Front React + Tailwind, API REST Node.js, opérations CRUD complètes. Conçu et livré en solo de bout en bout.

Stack front-end

  • Next.js
  • React
  • Tailwind CSS
  • Axios
  • Vite

Stack back-end

  • Express
  • MySQL
  • jsonwebtoken
  • Zod
  • argon2
  • Helmet
  • CORS
  • Nodemailer

Livré

  • Interface React + Tailwind CSS pour la consultation et l'édition des credentials (CRUD)
  • Communication front/back via Axios, configuration CORS côté serveur pour les requêtes cross-origin
  • API REST Express : routes authentifiées, opérations CRUD complètes sur les mots de passe
  • Authentification double facteur (2FA) — code envoyé par email via Nodemailer
  • Middleware d'auth basé sur jsonwebtoken + validation des payloads avec Zod
  • Hash des mots de passe utilisateurs avec argon2 (recommandation OWASP), headers HTTP durcis via Helmet
  • Architecture MVC : modèles, contrôleurs, routes séparés
